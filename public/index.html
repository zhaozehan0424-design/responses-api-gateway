快速开始

三步接入

无需改代码，把现有 OpenAI 客户端指过来就行。

注册 / 登录

使用类脑社区 Discord 账号注册或登录，在用户中心复制你的专属 API Key。

设置 Base URL

把客户端的 base_url 改为上面那条 /v1 地址。

开始调用

新项目优先调用 /responses； 通用 OpenAI 兼容客户端继续使用 /chat/completions。

cURL · 推荐 Responses API 
curl __BASE_URL__/responses \
  -H "Authorization: Bearer __API_KEY__" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "claude-haiku-4-5-20251001",
    "input": "你好，回复 OK 即可"
  }'
