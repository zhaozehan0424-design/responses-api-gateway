Relay Hub API Docs

Replace `https://your-domain.example/v1` with your deployed gateway base URL and `YOUR_DOWNSTREAM_KEY` with a downstream key configured on the server.

GET /v1/models

Returns the models visible to the authenticated downstream key.

POST /v1/responses

Recommended endpoint for new integrations and agent workflows.

POST /v1/chat/completions

Compatibility endpoint for existing OpenAI-style clients.

List Models

cURL 
curl https://your-domain.example/v1/models \
  -H "Authorization: Bearer YOUR_DOWNSTREAM_KEY"

Responses API

cURL 
curl https://your-domain.example/v1/responses \
  -H "Authorization: Bearer YOUR_DOWNSTREAM_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "claude-haiku-4-5-20251001",
    "input": "Reply with OK only.",
    "max_output_tokens": 16
  }'

Chat Completions

cURL 
curl https://your-domain.example/v1/chat/completions \
  -H "Authorization: Bearer YOUR_DOWNSTREAM_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "claude-haiku-4-5-20251001",
    "messages": [
      { "role": "user", "content": "Reply with OK only." }
    ],
    "max_tokens": 16
  }'

JavaScript Client

Node.js 
import OpenAI from "openai";

const client = new OpenAI({
  apiKey: process.env.RELAY_HUB_KEY,
  baseURL: "https://your-domain.example/v1"
});

const response = await client.responses.create({
  model: "claude-haiku-4-5-20251001",
  input: "Reply with OK only.",
  max_output_tokens: 16
});

console.log(response.output_text);

Deploy Checklist

  • Copy `.env.example` and set real server-side values.
  • Never expose `UPSTREAM_API_KEY` to users or client-side code.
  • Set `MODEL_ALLOWLIST` before sharing keys.
  • Set `RPM_LIMIT` and budget variables for shared deployments.
  • Use Vercel KV / Upstash Redis for durable logs and quotas.

Error Codes

HTTP Code Meaning
401 invalid_api_key Missing, invalid, disabled, or malformed downstream key.
403 model_not_allowed The selected model is not available to this key or group.
403 endpoint_not_allowed The selected endpoint is not enabled for this key or group.
402 quota_exceeded The key or Discord account has exhausted its budget.
429 rate_limit_exceeded The key exceeded its requests-per-minute limit.
502 upstream_request_failed The upstream model provider returned an error or could not be reached.
